# 📰 El Faro - Sitio Web HTML5

Proyecto desarrollado como parte de una actividad académica, que consiste en la creación de un sitio web para un periódico digital utilizando HTML5 y CSS.

## 📌 Características

- Estructura semántica en HTML5
- Diseño visual con CSS
- Uso de tipografía externa (Google Fonts)
- Layout con CSS Grid
- Efectos interactivos con :hover
- Diseño responsivo (adaptable a dispositivos móviles)
- Integración de contenido multimedia (video y audio)
